eduHUB Live

As we celebrate 15 years of impact at Global Study UK, I led the design and development of EduHUB Live â€” transforming the Arabian Awards and the Educators Networking Reception into an always-on platform that recognises counsellor excellence, shares practical resources and fosters regional collaboration across MENA and beyond.

Objective

A simple, elegant portal for company partners to reserve sponsorship packages for the next EduHUB Live event in December. The site focuses on a streamlined reservation flow and includes the event schedule with one-click additions to Google Calendar and Apple/ICal.

To maintain a controlled, professional partner experience, access is gated by email verification on first entry â€” this keeps attendee lists private and prevents public live previewing of the platform while ensuring only confirmed partners can view and book.

Every new access attempt and each booking triggers a notification to the organiser so outreach and follow-up remain timely and personal.

Design & Branding

The interface adopts a minimal, sophisticated luxury aesthetic: a flat UI with crisp layouts, generous spacing, and refined typography to communicate professionalism and premium value. Dark mode is the core visual theme â€” deep, tactile backgrounds contrast with subtle accent tones from the established palette.

Branding work (the logo, colour palette and carefully chosen fonts) was selected to reinforce the luxury vibe while remaining legible and approachable. Visual treatments are intentionally restrained so the focus stays on sponsorship content and the booking journey.

Development & Implementation

The platform is built on WordPress with Elementor for flexible content assembly, plus targeted custom code to meet access-control and notification requirements.

Custom PHP was added to the theme's functions.php to handle secure email verification and backend hooks; a JavaScript module in the theme folder manages the client-side verification flow and calendar interactions. The verified email list is stored as a JSON file in the secured WordPress backend and is checked at sign-in.

Verification runs once per device (using local storage and token checks) to reduce friction. New access and booking events send notification emails to the organiser including partner identity and selected package.

User Experience

The user journey is short and intentional: authenticate, review curated sponsorship options, and complete a focused booking with minimal distractions. The flat UI reduces cognitive load and highlights clear actions.

Email verification creates a trusted, partner-only environment while confirmation emails and organiser notifications close the loop transparently. Accessibility and responsiveness are built in so the booking path works reliably on desktop and mobile.

Overall the UX balances exclusivity with ease â€” partners feel welcomed into a premium, well-managed space and can make reservations quickly and confidently.
